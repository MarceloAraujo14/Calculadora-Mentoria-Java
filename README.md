Projeto Calculadora 

Objetivo: Desenvolver uma calculadora com 3 tipos de operações

Padrão
- Soma
- Subtração
- Multiplicação
- Divisão
- Potenciação

IMC
- Cálculo de imc

Área
- Área do circulo
- Área do quadrado
- Área do retangulo


Para rodar o projeto:
 Basta clonar e rodar pela IDE

Tecnologias:
Java 17, Gradle, Lombok, JUnit, Mockito
